Étant intéressés par le cinéma, notre projet consiste à mettre en place une plateforme qui permettra à un utilisateur de voir les différents films qui sont programmés dans une salle de cinéma et d’avoir la possibilité d’acheter un ticket. Pour cela nous avons implémenté une base de données pour stocker les différentes informations sur les films, les salles de cinéma, les utilisateurs de la plateforme etc. Ce site est un exemple de comment l'Istic pourrait utiliser une telle plateforme.
